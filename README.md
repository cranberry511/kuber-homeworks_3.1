## Решение задания

- 3 копии БД требуют 3 CPU и 12 ГБ RAM  
- 3 копии Кеша требуют 3 CPU и 12 ГБ RAM  
- 5 копий Фронтенда требуют 1 CPU и 0,25 ГБ RAM  
- 10 копии Бекенда требуют 10 CPU и 6 ГБ RAM

Суммарно 17 CPU и 30,25 ГБ RAM.  
Возможно расположить на 3 узлах. С учетом запаса для падения одной из node, каждый узел должен иметь возможность дежать половину нагрузку кластера. Также заложим базовую потребность для узла в 2 CPU и 4 ГБ RAM для не продуктивного окружения, и 4 CPU и 8 ГБ RAM для прода.
С учетом вышесказнного, оценка 3 узла по 11 CPU и 20 ГБ RAM для непродуктивного окружения, 13 CPU и 24 ГБ RAM для прода.
